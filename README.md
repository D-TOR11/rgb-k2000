### Dependencies

### Quick start


```bash
$ git clone git@github.com:D-TOR11/rgb-k2000.git
$ cd rgb-command
$ sudo pip3 install rpi_ws281x adafruit-circuitpython-neopixel
```

### Running command

```bash
$ sudo python3 pixel.py
```
